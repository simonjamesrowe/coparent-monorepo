# Tailwind Color Configuration

## Color Choices

- **Primary:** `teal` — Used for buttons, links, key accents
- **Secondary:** `rose` — Used for tags, highlights, secondary elements
- **Neutral:** `slate` — Used for backgrounds, text, borders

## Usage Examples

Primary button: `bg-teal-600 hover:bg-teal-700 text-white`
Secondary badge: `bg-rose-100 text-rose-800`
Neutral text: `text-slate-600 dark:text-slate-400`
